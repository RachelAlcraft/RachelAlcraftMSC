Some prelimary help for the distributions page

Histograms: enter Geo Calc X and Distribution A.
Note for contacts for distance distributions use e.g. C@SG-SG (or C@N-O, C@CA-CA, c@CB-CB)
Note for bounds the Upper is exclusive and the lower in inclusive
Scatter plots: enter Geo Calc X and Geo Calc Y, and Distribution A
The gradient can be entered in the gradient box, any numerical value, eg resolution, rfree, rvalue, residues, amino_no...
Probability Density: enter Geo Calc X and Geo Calc Y, and Distribution A
Compare Distributions: enter Geo Calc X and Geo Calc Y, and Distribution A and B for an overlay comparison
Compare Probability: enter Geo Calc X and Geo Calc Y, and Distribution A and B for an overlay comparison
Note for the amino code, enter a single 3 letter code, or a comma delim list, or ALL
Note that this is different for the distance distributions where ALL means all at the same time
A further restriction can be added for the sql, e.g. AND amino_code != 'PRO'

These geometric measures have been pre-calculated for non-similar high resolution structures from the pdb (<=1.3A) and a 2018-2019 set.
